# MemU Server

Standalone memory service with memorize and retrieve functionality.

## Features

- 🧠 **Memorize**: Store and process conversational memories
- 🔍 **Retrieve**: Query and retrieve relevant memories
- 🚀 **Async**: Built with FastAPI and async/await
- 🗄️ **PostgreSQL**: Vector similarity search with pgvector
- ⚡ **Temporal**: Reliable workflow orchestration
